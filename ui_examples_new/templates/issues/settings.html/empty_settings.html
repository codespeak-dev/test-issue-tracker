How to Set Up Integrations

Slack Integration:

  • Create a Slack app at https://api.slack.com/apps
  • Add the "chat:write" bot token scope
  • Install the app to your workspace
  • Copy the bot token and invite the bot to your desired channel

GitHub Integration:

  • Create a personal access token at https://github.com/settings/tokens
  • Grant "repo" permissions for private repositories or "public_repo" for public ones
  • Commit messages with "#<issue-id> Fixed" will automatically close issues
